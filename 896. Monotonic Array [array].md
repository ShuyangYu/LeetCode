# 896. Monotonic Array [array]

## My submission

```java
class Solution {
    public boolean isMonotonic(int[] A) {
        int l = A.length;
        if(l == 1){
            return(true);
        }
        
        boolean result_1 = true;
        boolean result_2 = true;
        
        for(int j = 0; j < A.length - 1; j++){
            if(A[j] < A[j + 1]){
                result_1 = false;
            }
        }
        
        for(int j = 0; j < A.length - 1; j++){
            if(A[j] > A[j + 1]){
                result_2 = false;
            }
        }
        return(result_1 || result_2);
    }
}
```

public boolean isMonotonic(int[] A) {
        boolean inc = true, dec = true;
        for (int i = 1; i < A.length; ++i) {
            inc &= A[i - 1] <= A[i];
            dec &= A[i - 1] >= A[i];
        }
        return inc || dec;
    }
